# Project Name: RAA - Recruit An Apprentice
Recruit an Apprentice is a service that helps employers and providers(on behalf of employers) post and manage apprenticeship vacancies. It allows businesses to advertise roles and manage applications. 
Vacancies are listed on **Find an Apprenticeship**(FAA), connecting employers with potential apprentices. 

**Pipeline URL** - https://sfa-gov-uk.visualstudio.com/Digital%20Apprenticeship%20Service/_release?definitionId=315&view=mine&_a=releases
