**Sign up to confirm your apprenticeship details (CMAD)**

Service portal that enables aprrentices to confirm their apprenticeship details and future amendments as and when required

**Prerequisites to run the tests**:
* General prerequisities can be gathered from the Main README.md file (https://github.com/SkillsFundingAgency/das-enterprise-automation-test-suite/blob/master/README.md)
* Apprentice Commitments User secrets are set up (Please refer to the main README.md)
