# Find apprenticeship training for your apprentice (FAT)
Platform for employers to find training courses and providers for their apprentices.

## Prerequisites to run the tests:
1. General prerequisities from the Main read me
2. FAT User secrets set up (Please refer to the main read me)

## Pipelines related to the project
* Regression Automation SFA.DAS.FAT.UITests
* P1_das-fat-v2-automation-test-suite
* Integration API :- das-roatp-api
* Outer API :- das-apim-endpoints-findapprenticeshiptraining
* Web :- das-findapprenticeshiptraining
* Database :- das-{env}-crs-db
